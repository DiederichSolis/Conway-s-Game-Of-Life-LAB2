

RUNNING DE PROJECT
<p>
1.) clonar repositorio
</p>

```
git clone https://github.com/DiederichSolis/Conway-s-Game-Of-Life-LAB2.git
```

<p>
2.) Pasarse a la carpeta game
</p>

```
cd game
```

<p>
3.) Realizar el build
</p>

```
cargo build
```

4.) Correrlo
</p>

```
cargo run 
```
